  • Github Connection

    December 16, 2023

    在连接github时，经常会遇到速度慢或者无法连接的情况，这是在cmd命令窗口下使用ping命令找出具体的Ip地址，把这些显示的IP地址添加在本地的hosts文件中，可以一定程度缓解连接github的问题

  • Windows局域网共享打印机

    首先把要访问的印机设置为共享打印机。然后启用网络发现、启用文件和打印机共享这几个选项。最后在控制面板的管理工具中，打开本地策略，点击【安全选项】，在右侧底部列表中选择开启guest。

  • Getting Started With Web
