#### Getting  started with  web

##### HTML

HyperText  Markup  Language ,用标签表示的的Element来结构化网页。常见的标签表示元素有

~~~
<html>   </html>
<head>   </head>
<body>   </body>
<h1>    </h1>
<p>     </p>    --paragraph
<table>  </table>
  <tr>   </tr>
  <th>   </th>
  <td>   </td>
<form>  </form>
  <lable>
  <input  type="text">
  <input   type="checkbox">
  <input  type="button">
 ... 


~~~

CSS

Casecade  Style  Sheet ，主要有两个功能：

* Style  Html  Element 

* UI  Layout 

  HTML页面上的每一个Element都是一个盒子，基于盒模型进行页面布局。

JAVASCRIPT

通过向HTML文件中添加`<script>  </script>`javascript代码，可以创建动态网页。比如从后台数据库获取数据，和少量的HTML静态网页模板一起就可以维护大量数据的动态网页。
