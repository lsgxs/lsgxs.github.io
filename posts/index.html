Usehugo

使用hugo建立静态站点 hugo 是一款简单且功能强大的静态站点生成器，让新手很快上手制作自己的静态博客，下面是具体的使用方法要点： step1: 下载hugo可执行文件，设置paht环境变量 下载windows平台的hugo可执行文件,把hugo可执行文件所在目录添加到path环境变量.建立一个保存hugo站点的目录，比如g:\hugo\ 在g:\hugo\bin目录下保存hugo.exe可执行文件；在g:\hugo\sites下保存站点。具体目结构如下 g:\hugo\bin\hugo.exe g:\hugo\sites step2: 新建站点 切换到g:\hugo\sites目录下，在cmd窗口下按顺序运行下面的语句 hugo new site quickstart --创建站点 cd quickstart git init --用git init 初始化站点，纳入git管理，在接下来添加theme就可以使用git命令了。这条语句会在当前目录下新建一个.git目录及相关的文件。 step3: 添加主题 (themes) git submodule add https://github.com/hugo-theme-xx.git themes/hugo-theme-xx --下载自己喜欢的theme，这里是通用写法 将 theme = ‘hugo-theme-xx’添加到config.toml --在配置文件中标注theme step4：hugo new posts/my-first-post.md ----创建markdown格式的文档，文件会保存在项目的content\posts目录下面 关于hugo的下生成的markdown格式的文件头，可以加上featured_image: "/images/xxx.png",其中这个images目录是可以自己在项目的static目录下建立， 然后把需要显示在文章边栏的图像文件保存在static/images目录下 step5: 自定义主题和配置文件 Open up config.toml in a text editor: baseURL = "https://username.github.io/" languageCode = "en-us" title = "My New Hugo Site" theme = "ananke" Replace the title above with something more personal.
