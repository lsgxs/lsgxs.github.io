机油认证标准简明笔记（完整版 · 整合理解）

0. 机油桶上常见的四大认证（总览）

  • SAE：规定 粘度等级（稠不稠）。例：5W-30。
  • API：规定 质量等级（美标，好不好）。例：SP。
  • ACEA：规定 性能与环保等级（欧标，更严）。例：A5/B5, C3。
  • ILSAC：规定 节能与适配等级（日美系专属）。例：GF-6A。

1. SAE 粘度等级

  • 定义：机油的流动与粘稠程度，分低温粘度和高温粘度。
  • 格式：5W-30
    • 5W：冬季低温粘度，数字越小，低温流动性越好。
    • 30：100℃ 高温粘度，数字越大，油膜越厚。
  • 结论：SAE 规定“稠不稠”。

2. API 性能标准

  • 定义：美国标准的机油“毕业证书”，侧重清洁和抗磨损。
  • 常见类型：SN / SP，SP 优于 SN。
  • 结论：API 规定“好不好”。

3. ACEA 性能标准（重点扩展）

核心理解（务必记住）

C2 ≈ A5/B5 的低灰版：从 A5/B5（高灰分）进化到 C2（低灰分），性能取向不变（低粘度、高节能），但灰分降低以保护国六车的颗粒捕捉器（GPF）。
C3 ≈ A3/B4 的低灰版：从 A3/B4（高灰分）进化到 C3（低灰分），性能取向不变（高粘度、高保护），但灰分降低以保护 GPF。

简单说：C 系列不是另起炉灶，而是把 A/B 系列的性能配方“降灰分”后，用于带 GPF 的国六车。

3.1 A/B 系列（高灰分，适配国五及以下无 GPF 车型）

标准HTHS（高温高剪切）灰分特点适用车型
A3/B4≥ 3.5 cP≤ 1.6%高粘度、高保护、抗磨强德系老车、高负荷涡轮
A5/B52.9–3.5 cP≤ 1.6%低粘度、高节能、兼顾保护国五及往日系/美系/部分欧系

3.2 C 系列（低灰分，适配国六带 GPF 车型）

标准HTHS（高温高剪切）灰分特点适用车型
C2≥ 2.9 cP≤ 0.8%低粘度节能 + 低灰分国六日系/美系带 GPF（A5/B5 的低灰进化版）
C3≥ 3.5 cP≤ 0.8%高粘度保护 + 低灰分国六德系/欧系带 GPF（A3/B4 的低灰进化版）
C52.6–2.9 cP≤ 0.8%超低粘度 + 低灰分新一代国六车（宝马、沃尔沃等低粘度车型）

4. ILSAC 节能标准

  • 定义：API 的“节能加强版”，日美系量身定制。
  • 常见写法：GF-6A（星爆标志，常见粘度）、GF-6B（盾牌标志，仅 0W-16）。
  • ** 重要澄清**：GF-5/GF-6 与“国五/国六”毫无关系！
  • GF-6 vs GF-5：GF-6 全面升级，国五无 GPF 车优先选 GF-6A，向下兼容。

5. 重点

  1. 首看车辆手册。手册写什么认证/粘度，就用什么。
  2. 国六b + 涡轮 → 必须用 低灰分机油（ACEA C 系列），不要用 A5/B5。
  3. 国五及以下 + 涡轮 → 可用 A5/B5 或 GF-6A，两者都不错。
  4. C2 = A5/B5 的低灰进化版，C3 = A3/B4 的低灰进化版。性能取向相同，灰分不同。
  5. GF-5 ≠ 国五，GF-6 ≠ 国六。它们是机油性能标准，不是排放标准。
  6. 国五不带颗粒捕捉器的车，推荐优先选 GF-6A（比 GF-5 好，向下兼容）。

1、sae 5w30 或者0w30 机油粘度

2、api sn/sp 机油品质

3、acea a5/b5 c2; a3/b4 c3 机油性能和环保特点

​ a5/b5 是高灰分的低粘度机油，适合国五标准汽车；c2则是进化的低灰分低粘度机油，适合国六标准

a3/b4 是高灰分高粘度的机油，适合国五标准汽车；c3则是进化的低灰分高粘度机油，适合国六标准

a/b系列是高灰分机油，适合国五标准汽车，c系列则是低灰分机油，适合国六汽车。

4、ilsac gf-6

简答总结：

国六带颗粒捕捉器 → ACEA C 系列（低灰分） 国五不带颗粒捕捉器 → ACEA A/B 系列（高灰分） 粘度选择看发动机手册

定期使用壳牌机油清洁发动机

恩渡和壳牌各有优势，正好形成了一套科学的发动机养护方案：

    日常养护（恩渡）：负责长效稳定地维持发动机状态。它的长效设计能抑制油泥产生，确保发动机长期运转在一个非常干净、稳定的状态。
    
    定期深度清洁（壳牌）：负责介入强化清洁。壳牌以出色的清洁能力著称，其特别的清净分散剂能有效清除油泥积碳，保持内部清洁。

就像日常搞好卫生保持整洁，但偶尔也请一次专业家政团队来做大扫除一样。将两种优势结合，是保持发动机巅峰状态的理想方案。


📅 具体执行方案：多久一次

考虑到这款发动机、用油品质以及大家的经验，建议每行驶 3 年或 4-6 万公里，进行一次壳牌机油的深度清洁。

选用时机建议在保养同时进行，操作步骤很简单：

    放掉旧机油 -> 加入壳牌机油 -> 正常使用5000-8000公里后 -> 再换回恩渡机油继续日常养护。

此外，日常中注意这几点会更好：

    每次保养都换机滤：不管是换哪种油，机滤都要一起换掉。
    
    固定周期使用：把这次清洁纳入长期保养计划，比如计划好“3次恩渡 + 1次壳牌”这样的策略。
    
    保持汽油品质：高品质的汽油对减少积碳、保持引擎清洁也非常重要。
    
    推荐壳牌型号：具体型号建议选择“壳牌超凡喜力 5W-30 ACEA A5/B5”这个粘度和标准。
